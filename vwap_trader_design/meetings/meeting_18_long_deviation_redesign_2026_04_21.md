## 3. A 제안 (2026-04-21)

**선택**: X2 (σ 척도: std → ATR) 단독 + X4 일부 (low → close)
**배수**: -2.0 고수 (Q3-final 경계 존중)

### 세부 정의

```python
# 현행 (engine.py:627 + module_a.py:153)
sigma_1 = std(typical_price) over 24 × 1H bars
threshold_long = daily_vwap + (-2.0) * sigma_1
trigger: any(low[i] < threshold for i in last 3 bars)

# 제안
atr_14 = ATR(14) on 1H bars  # Wilder's smoothing, true range
threshold_long = daily_vwap + (-2.0) * atr_14
trigger: any(close[i] < threshold for i in last 3 bars)
```

### 변경점
- σ 척도: `std(typical_price, 24)` → **`ATR(14)`**
- 배수: **-2.0 유지**
- 이탈 기준: **`low` → `close`** (wick-only trigger 제거)
- VWAP 기준: daily 유지

### 근거 (A)
1. `std(typical_price, 24)`는 drift 성분 포함 → σ 팽창 → -2σ 도달 불가 (S2 98.9% 차단의 수학적 원인)
2. ATR은 봉간 true range만 측정 → 순수 노이즈, 평균회귀 전제에 정합
3. Brian Shannon(A 스승) 정본 VWAP 편차 밴드 = std 아닌 ATR 배수 기반
4. 배수 2.0 고수로 "정상 노이즈 2배 이탈" 통계 원칙 보존

### 예상 효과
- 진입 빈도: 3~5배 증가 (drift 제거 후)
- 품질: close 기준이 wick-only 스침 노이즈 차단
- Trade-off: flash crash 시 ATR 즉각 확장 → 자동 방어 (자기조절)
- no_deviation 예상: 98.9% → 60~75%

### p-hacking 경계 (A 자체 선언)
- σ 척도 원리적 교체 (drift 분해) — Q3-final 범위 내
- 배수 -2.0 불변 → 파라미터 스캐닝 아님
- 통계학 근거 (drift 오염 제거), 결과 맞춤 아님

**A 핵심 선언**: "σ가 차단하는 게 아니라, 잘못된 σ가 차단한다. 척도를 바로잡으면 원래의 평균회귀 빈도가 돌아온다."

## 7. A 반증 조건 문서 (2026-04-21, F Q2 의무 이행)

**범위 고정**: X2·X4 채택 확정분에 대한 사망 조건만. 배수·대체 척도 논의 제외.

### 7-(a) 대안 가설 (drift 오염 외)

- **H1. 변동성 군집 (Volatility Clustering / GARCH lag)**: ATR(14) 단순 평균 → regime 전환에 측정 lag. 저→고 변동성 전이 구간 ATR 과소 측정으로 `-2×ATR` 임계가 얕게 형성될 수 있음. "미도달"의 원인이 drift가 아니라 척도 lag일 수 있음.
- **H2. 세션 편향 (Time-of-day compression)**: Asian session 구조적 저변동성 (24h RV 대비 55~70%). 24h 기준 ATR 임계가 세션-측정창 불일치로 Asian session에서 체계적 미도달 가능. drift 무관한 정규화 오류.
- **H3. Fat tail 수축 (시장 미시구조 변화)**: 2023년 이후 HFT/패시브 확대로 BTC/ETH 5M 수익률 kurtosis 추세 하락 가능. 정규분포 기반 -2σ 빈도 자체가 경험적 희소해진 것일 뿐 drift 무관.

### 7-(b) 가설별 무효화 관측 기준

| 가설 | 참이면 | 무효(기각) |
|---|---|---|
| H1 | regime 전환 구간(RV5 전기 +30%↑) ATR(14)의 RV5 수렴 lag ≥ 10봉, signal count ≤ 안정 구간의 50% | lag ≤ 3봉 또는 signal count ≥ 80% |
| H2 | 세션별 평균 ATR 정규화 후 Asian 터치 빈도 ≤ US의 60% | 정규화 후 세션 편차 ≤ ±15% |
| H3 | 5M kurtosis rolling slope 음수, \|t-stat\| > 2, 최근 12개월 평균 kurtosis < 4 | \|t-stat\| ≤ 1 또는 최근 12개월 평균 kurtosis ≥ 5 |

**H1·H2·H3 모두 기각**되면 drift 오염 가설이 소거법으로 유일 후보 승격.

### 7-(c) ATR(14)+close 폐기 임계값

아래 중 **어느 하나라도** 도달 시 → ATR(14)+close 폐기, std+low 원형 복귀.

| # | 지표 | 폐기 기준 | 비교 대상 | 관측 기간 |
|---|---|---|---|---|
| C1 | 순 EV 델타 (R/trade) | ATR+close가 std+low 대비 **−0.15 R 이상 열위** | 동일 기간·심볼 병렬(실거래 vs 섀도우) | 100거래 또는 90일 중 먼저 |
| C2 | 승률 | ATR+close 승률 **< 55%** (목표 60% −5%p) | 절대 기준 | 최근 60거래 rolling |
| C3 | 신호 빈도 | 월평균 signal count **−40% 이상 감소** | 직전 6개월 std+low 월평균 | 3개월 연속 지속 |

### 7-(d) 운영 규칙

1. **병렬 운영**: 2026-04-21 ~ 최소 90일간 ATR+close 실거래, std+low 섀도우 기록
2. **리뷰 주기**: 2주 간격. 임계 도달 시 즉시 폐기 안건 상정 ("한 사이클 더 보자" 불가)
3. **폐기 후 복귀**: std+low 원형으로만 복귀. 제3의 척도(Keltner, Donchian 등)는 별도 절차
4. **임계 미도달 시**: 본 문서가 사후 정당화 차단 장치로 고정. ATR+close 유지 자동, 재논의 불요

**A 서명문**: "척도를 바꾼 순간 이전 세계의 분포로는 돌아갈 수 없다. 새 척도의 사망 조건은 새 척도가 살아있는 동안 고정한다. — 이것이 본 문서의 유일한 존재 이유다."
