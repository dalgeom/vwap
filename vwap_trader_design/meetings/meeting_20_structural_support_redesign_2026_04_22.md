### Q1. 재설계 옵션

구조적 문제를 먼저 명확히 한다. `structural_support`가 0.0%인 이유는 코드 버그가 아니라 물리적 불가능성이다.

Volume Profile의 HVN·POC·VAL은 정의상 거래량이 집중된 지점이다. 그런데 VWAP−2×ATR 극단은 정의상 LVN 구간이다. 고거래량이 저거래량 구간에 존재하리라는 가정 자체가 모순이다. 이 구조적 모순은 기준점을 low→close로 바꿔도 해소되지 않는다. C metric 0.0% 불변이 그 증거다.

따라서 재설계 방향은 "VP 레벨을 어떻게 더 잘 찾느냐"가 아니라 "극단 이탈 구간에서 평균회귀 진입 근거를 어떤 다른 방식으로 확인하느냐"다.

---

#### Option P3-1: 근접 임계값 확장 (0.5×ATR → 1.5×ATR)

```python
# 변경 전
near_val = abs(deviation_ref - vp_layer.val) <= 0.5 * atr
near_poc = abs(deviation_ref - vp_layer.poc) <= 0.5 * atr
near_hvn = any(abs(deviation_ref - hvn) <= 0.5 * atr for hvn in vp_layer.hvn_prices)

# 변경 후
near_val = abs(deviation_ref - vp_layer.val) <= 1.5 * atr
near_poc = abs(deviation_ref - vp_layer.poc) <= 1.5 * atr
near_hvn = any(abs(deviation_ref - hvn) <= 1.5 * atr for hvn in vp_layer.hvn_prices)
```

**근거**: 현재 임계값(0.5×ATR)이 과소한지, 아니면 레벨 자체가 구조적으로 부재인지 구분이 필요하다. 임계값 확장은 가장 작은 변경으로 두 가설을 분리하는 방법이다. 신규 지표 없음. 코드 변경 최소.

**한계**: 1.5×ATR이 "구조적 지지 근처"를 너무 넓게 정의하면 조건이 의미를 잃는다. n≥30 후 hit rate가 여전히 낮으면 레벨 부재(H2) 확인.

---

#### Option P3-2: VAL 하방 이탈 조건으로 개념 전환

```python
# near_val 교체 (near_poc/near_hvn 유지)
# 새 조건: deviation_ref가 VAL 아래에 있되, VAL에서 1.0×ATR 이내
below_val_zone = (vp_layer.val - 1.0 * atr) <= deviation_ref < vp_layer.val
structural_support = below_val_zone or near_poc or near_hvn
```

**근거**: 평균회귀 관점에서 `near_val`(VAL 근접)보다 `below_val`(VAL 이하)이 더 강력한 신호다. Brian Shannon의 논리: "VAL 이하에서 거래된다는 것은 전체 거래량의 상위 70%가 현재가 위에 있다는 뜻이다 — 이것이 평균회귀의 구조적 인력이다." 점 근접 → 구간 멤버십으로 개념 전환.

**전제 가설**: VWAP−2×ATR 이탈 시 deviation_ref < vp_layer.val이 항상 성립하지 않아야 조건이 의미를 갖는다. 신규 지표 없음 (vp_layer.val 기존 사용 중).

#### Option P3-3: 4H Swing Low 구조적 지지 교체

```python
# VP 레벨 대신 최근 4H 스윙 로우를 구조적 지지로
swing_low_4h = min(c.low for c in candles_4h[-10:])  # 최근 10봉(40H)
near_swing_low = abs(deviation_ref - swing_low_4h) <= 1.0 * atr
structural_support = near_swing_low or near_poc or near_hvn
```

**근거**: VP 이론 레벨보다 실제로 매수가 발생한 가격(swing low)이 더 직접적인 구조적 지지다. Paul Rotter — "가격이 이전에 멈춘 곳은 다시 멈추려는 경향이 있다." HFT·마켓 메이커 모두 스윙 로우를 참조한다. 이론적 레벨(HVN)보다 경험적 레벨(swing low)이 LVN 구간에서 더 유효하다.

**신규 지표**: `4H swing low (10봉 기준)` — 기존 미사용. B.5 ②③ → Q3에 포함.

### Q1. VAL 자기참조 위험 여부

**결론: 자기참조 무력화 위험 낮음. 그러나 구조적 퇴화 위험은 별도 경로로 실재한다.**

`compute_volume_profile(candles_168h)`는 순수 함수다. 168H 캔들 리스트를 받아 정적인 `VolumeProfile`을 반환하며 내부에 상태가 없다. `vp_layer`는 B.1 진입 로직에 외부 파라미터로 주입된다. 따라서 VP 계산 자체가 진입 조건 안에서 실시간 피드백 루프를 형성하지 않는다.

B가 제기한 "VAL이 deviation_ref를 쫓아 하방 이동" 시나리오는 이론적으로 가능하나, 168H 중 극단 이탈 캔들은 1~3개에 불과하다. VAL은 168개 캔들의 70% 거래량이 집중된 구간의 하단이므로, 소수 극단 캔들이 VAL을 유의미하게 끌어내리는 충격은 구조적으로 제한된다.

그러나 더 심각한 위험은 별도 경로에 있다. Accumulation 레짐에서 VWAP−2×ATR이 구조적으로 VAL보다 낮다면 — Accumulation 레짐 특성상 이 가설은 충분히 성립할 수 있다 — `deviation_ref < vp_layer.val`이 극단 이탈 시 **거의 항상 True**가 된다. 이것이 실질적인 퇴화 경로다. 자기참조가 아니라 **구조적 동어반복** 퇴화다.

A의 반증 조건 P3-2 (a): `below_val_zone=True 비율 > 90% (n≥10)` — 이 조건이 이 위험을 정확히 탐지한다.

### Q2. VP 이론 정합성 순위

**순위: P3-2 > P3-3 > P3-1**

**P3-2 (Below VAL) — 최우선**

Dalton의 Value Area 이론에서 VAL은 단순한 수치가 아니다. "전체 거래의 70%가 VAL 위에서 일어났다"는 사실 자체가 구조적 인력을 만든다. 가격이 VAL 이하로 이탈하면, 시장 참여자 70%의 거래 포지션이 현재가보다 위에 있는 상태가 된다. 이것이 평균회귀 인력의 VP 이론적 근거다. `below_val_zone`은 "점 근접" 휴리스틱이 아니라 **구간 멤버십**으로 개념을 전환한다는 점에서, Dalton이 정의한 VA 이론과 가장 직접적으로 정합한다.

**P3-3 (4H Swing Low) — 차순위**

Wyckoff의 Spring 개념과 일치한다. Accumulation 국면에서 구조적 지지를 이탈한 직후 반전하는 패턴이다. VP 이론 직계는 아니지만, Wyckoff Accumulation 체계 내에서 경험적으로 검증된 구조다. LVN 구간에서 이론적 레벨보다 경험적 레벨이 더 유효할 수 있다는 A의 판단을 지지한다.

**P3-1 (1.5×ATR 확장) — 최하위**

VP 이론과 무관하다. "VAL/POC/HVN에 더 넓게 근접"이라는 개념은 이론적 정당화가 없다. Dalton에게 "VAL에서 1.5 ATR 떨어진 가격도 VAL 근처"라는 논리는 Value Area 정의 자체를 해체한다. 임계값 조율이지 VP 이론이 아니다.

### Q3. LVN 대안 실현 가능성

**결론: VP 관점에서 LVN 근접을 "지지 조건"으로 사용하는 것은 이론적으로 역방향이다. 채택 불가.**

LVN(Low Volume Node)은 거래량 공백 구간이다. 경매 이론(Steidlmayer)에서 LVN의 의미는 하나다 — **시장이 이 가격대를 '공정하지 않다'고 판단하고 빠르게 통과했다.** LVN은 지지/저항이 아니다. 지지/저항은 거래가 많이 일어난 곳(HVN, POC)에서 발생한다.

A의 §1 진단 — "VWAP−2×ATR 극단은 정의상 LVN 구간이다" — 은 정확하다. 그런데 이 LVN 구간을 이번엔 *지지 조건*으로 쓰자는 것은 모순이다. LVN에 근접했다는 사실은 오히려 가격이 빠르게 통과할 것임을 시사한다.

**LVN이 극단부에 구조적으로 존재할 가능성**: 예, 존재한다. 그것이 문제의 핵심이다. 극단부는 *항상* LVN 구간이다. 따라서 "극단부 LVN 근접" 조건은 극단 이탈 시 거의 항상 True가 된다 — P3-1의 퇴화 위험과 동일한 경로다.

단, VP 이론적으로 유효한 해석은 존재한다: 가격이 VAL 아래 LVN을 통과하고 있다면, **Value Area로의 복귀 인력이 작동하는 국면**이다. 그러나 이것은 LVN 자체가 지지인 것이 아니라, VAL이 위에서 인력원이 되는 구조다. 이 개념은 P3-2가 이미 포착한다. LVN을 조건에 명시적으로 추가하는 것은 P3-2의 재진술이거나, 더 위험한 해석이다.

P3-3의 4H Swing Low가 LVN 대안보다 우월한 이유: Swing Low는 *실제 매수가 발생했던 가격*이다. LVN은 매수가 일어나지 않았던 구간이다.

> **(나)** 39개월 데이터에서 0.5×ATR / 1.0×ATR / 1.5×ATR 세 값의 below_val_zone=True 발동 비율 비교표 제출.

#### CONDITIONAL 3 — 공통 WR 통과 기준 55%

"55%"는 "수익 우위(기대값 양수)"를 위한 최소선이지만, 통계적 유의성과는 다른 기준이다.

n=30, WR=55%(≈16.5/30 → 17회 이상)일 때 이항검정 기준 p-value (귀무: p=0.5):  
P(X≥17 | n=30, p=0.5) ≈ 0.29

p≈0.29는 통계적으로 유의미하지 않다(p<0.05 미달). 즉 n=30에서 55% WR은 우연에 의한 결과를 배제하지 못한다. n=30에서 통계적 유의성(p<0.05)을 위해 필요한 WR은 약 63%(≈19/30) 이상이다.

두 가지 선택이 있다:  
**(가)** WR 55% + n=30을 "모듈 A 철학 기준(기대값 우위)"으로 명시적으로 정의 — 통계적 유의성을 요구하지 않는다는 의미를 F가 판결문에 명시.  
**(나)** 통계적 유의성을 요구한다면 n 기준 상향(≥50) 또는 WR 기준 상향(≥63%, n=30 기준).

이것은 이 회의가 결정할 사항이다. 그러나 정의 없이 "55%"를 통과 기준으로 채택하면 동일 숫자가 서로 다른 해석 하에 사용될 위험이 있다.

**F에게 부과하는 CONDITIONAL 조건**:

> F는 최종 판결에서 WR 55% 기준이 다음 중 어느 정의를 채택하는지 명시할 것:  
> **(가) 철학적 기준**: "기대값 양수를 위한 최소 WR. 통계적 유의성은 n 증가 후 별도 검토 대상." — 현행 55% + n=30 유지.  
> **(나) 통계적 기준**: "p<0.05 귀무 기각 요건 충족." — n=30 기준 WR≥63%, 또는 WR=55% 유지 시 n≥50으로 상향.  
> 정의 명시 없이는 "55% 통과"의 의미가 회의마다 재해석될 수 있다. D는 어느 쪽이든 선택 자체를 막지 않는다. 다만 선택하지 않은 채 넘어가는 것을 막는다.
