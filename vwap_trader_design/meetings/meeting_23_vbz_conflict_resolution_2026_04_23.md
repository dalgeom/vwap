## §A — 설계 제안 (박정우, 2026-04-23)

### 방향 B: VBZ 시점 분리
- N=5, M=3 (이탈 봉 직전 5봉 중 3봉 이상 VBZ)
- VBZ 체크 시점: 조건 1 이탈 봉 [t] 직전 [t-5 ~ t-1]
- 이탈 봉 자체는 체크 대상 제외
- 조건 2 below_val_zone 우려: near_poc / extreme_exhaustion 의존은 "신호 정화 효과"로 해석
- 실패 조건: VBZ → 즉각 추세 전환 / N봉 경과 후 균형 소멸 / M=3 느슨함

### 방향 C: VPP (VWAP Proximity Pre-condition)
- 조건: 이탈 봉 [t] 직전 K=8봉 중 J=5봉 이상에서 |close_i - VWAP_i| ≤ 1.0×ATR_i
- 시간 분리 구조: [t-8 ~ t-1] vs [t, t+1] → ESC-001/002 충돌 구조 부재
- 실패 조건: 저유동성 허위 VPP / ATR 급등 직전 구간 / K 구현 오류

## §C — VP 검토 보고 (이지원, 2026-04-23)

### [질문 1] 57.2% 추세 구간 VP 필터 가능성
**가능. POC drift + VA slope 두 지표로 Balance/Imbalance 구분.**

- **Δ_POC_3d** = POC_7d(t) − POC_7d(t-72봉): `|Δ_POC_3d| < α×ATR` (α: 0.8~1.5)
- **VA_slope_3d** = VA_mid(t) − VA_mid(t-3일): `|VA_slope_3d| < β×ATR×72` (β: 1.0~2.0)
- 두 조건 AND → Balance 환경 확인 → 평균회귀 유효 판정
- 이론 근거: POC가 이동 중이면 시장은 새 fair value 탐색 중 → 평균회귀 전제 성립 불가

### [질문 2] VPP + VP 보완 조합
- VPP(K=12, J=4) + `|Δ_POC_3d| < α×ATR`: 추정 **2.5~3.0건/일**
- 단독 VPP(K=8,J=5) 2.11건/일 대비 개선되나 6건/일 달성 여전히 불확실

### [질문 3] VP 독립 제안 — VP-PMF (POC Magnet Filter)

**조건 정의:**
```
PMF-1: POC_7d > close_t                        (POC가 위에 존재 = 회귀 목표 명확)
PMF-2: (POC_7d − close_t) ≤ γ×ATR(14,1H)     (γ: 2.0~3.0, 회귀 거리 현실적)
PMF-3: |Δ_POC_3d| < α×ATR(14,1H)             (α: 0.8~1.5, POC 안정 = Balance)
```
TP1 = POC_7d (near_poc 논리와 일관)

**G 프로토콜 자가 체크 (C가 직접 수행):**
| 조건 | 교집합 존재 | 판정 |
|---|---|---|
| 조건 1 (close < VWAP-2σ) | close < VWAP-2σ ⊆ close < POC_7d | ✅ |
| 조건 2 (near_poc/VAL/HVN) | PMF-2 범위가 near_poc보다 넓음, 보완 관계 | ✅ |
| 조건 3 (반전 캔들) | VP와 무관, 독립 | ✅ |
| 조건 4 (RSI ≤ 38) | VP와 무관, 독립 | ✅ |
| 조건 5 (volume > MA20×1.2) | PMF 거래량 조건 없음, ESC-002 구조 부재 | ✅ |

**예상 빈도**: Condition 1 4.94건/일 × PMF 통과율(추정 40~55%) = **약 2.0~2.7건/일**

**C 의견**: 57.2%는 평균회귀 유효성 붕괴가 아니라 필터 부재의 문제.
Balance 42.8% 구간에서 평균회귀는 작동한다. POC drift가 그 구분 도구다.
