## 목적

Volume Balance Zone(VBZ) 레짐 조건이 BTC/ETH 에서 일평균 충분한 발동 빈도를 보이는지 실증.
PASS 시 VBZ를 레짐 필터로 채택, FAIL 시 폐기.

---

## Regime 조건 (명세 그대로 구현, 임의 수정 금지)

```
VAL(7일 VP) ≤ close ≤ VAH(7일 VP)
AND volume_1H < MA(volume, 20) × 0.8
```

- `VAL`: 7일 Volume Profile 기준 Value Area Low
- `VAH`: 7일 Volume Profile 기준 Value Area High
- `volume_1H`: 현재 1시간봉 거래량
- `MA(volume, 20)`: 직전 20봉 거래량 단순 이동평균

### 경계 처리 (strict, C.Q1 확정)

```
close < VAL → 즉시 regime 이탈 (strict)
```

VAL 하단 이탈 시 조건 불충족으로 즉각 처리. 버퍼 없음.

## PASS / FAIL 기준

| 판정 | 조건 |
|---|---|
| **PASS** | 일평균 VBZ regime 발동 ≥ 6건 |
| **FAIL** | 일평균 < 4건 |
| **경계** | 4건 이상 6건 미만 → 조건 1~5 실측 통과율 병행 보고 필수 |

---

## 추가 보고 의무 (C-22-4)

대상 구간: VA 계산 기준일로부터 **3일 이상 경과한 구간**만

보고 항목:
- VBZ 발동 후 **이탈 지속** 비율 (close가 VA 밖으로 나간 후 회귀 없이 지속)
- VBZ 발동 후 **회귀** 비율 (close가 VA 안으로 복귀)
- 두 비율 **분리 보고 필수**

판단 기준:
- **이탈 지속 > 회귀** 시: **즉시 의장(Dev-PM) 보고** → VBZ 전제 붕괴 트리거 발동

---

## 병목 분석 의무

보고서에 다음 항목 명시:
- VA 범위 조건(`VAL ≤ close ≤ VAH`)이 단독으로 발동 차단한 비율
- 거래량 조건(`volume_1H < MA(volume,20) × 0.8`)이 단독으로 발동 차단한 비율
- 양 조건 동시 차단 비율

## 분리 메트릭 요건 (C-22-5 의무 — 누락 시 즉시 반려)

보고서는 다음 축으로 메트릭을 **반드시 분리** 출력:

### by-year
- 2021, 2022, 2023, 2024, 2025~2026 각각 별도 행

### by-regime (시장 국면)
- 강세장 구간
- 폭락 구간
- 회복 구간

각 분리 셀에 포함할 항목: 발동 건수, 일평균 발동 수, (경계 구간 해당 시) 조건 1~5 통과율
