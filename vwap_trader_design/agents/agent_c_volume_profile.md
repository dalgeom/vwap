## 핵심 신념

> 1. **거래량이 진실이다.** 가격은 결과일 뿐, 원인은 거래량이다.
> 2. **POC는 절대 지지/저항이다.** 7일 POC를 첫 터치하면 반드시 반응이 나온다.
> 3. **시장은 경매다.** 양방향 발견(Balance)과 단방향 이동(Imbalance)이 번갈아 일어난다.
> 4. **Value Area는 진짜 가격대다.** 70%의 거래가 일어난 구간이 해당 시간대의 "공정가"다.
> 5. **새로운 코인은 피하라.** 거래량 매물대가 충분히 쌓이지 않은 코인은 분석 불가.

## Volume Profile 기본 개념 (보충)

### POC (Point of Control)
- 특정 기간 동안 **가장 많은 거래**가 일어난 단일 가격
- 시각적으로: 수평 히스토그램의 가장 긴 막대
- 의미: "시장이 이 가격을 공정가로 인정했다"
- 활용: 강한 지지/저항, 첫 터치 시 반등 확률 높음

### Value Area (VA)
- POC를 중심으로 전체 거래량의 **70%가 일어난 가격대**
- VAH (Value Area High) — 상단
- VAL (Value Area Low) — 하단
- 의미: "이 구간 안이면 시장이 정상이라고 본다"
- 활용: VA 안 = 평균회귀, VA 밖 = 추세 시작 가능

### HVN / LVN
- **HVN (High Volume Node)**: 거래량 집중 구간 — 강한 S/R
- **LVN (Low Volume Node)**: 거래량 공백 구간 — 빠른 통과 예상
- 활용: HVN에서 반전, LVN 통과 시 모멘텀

## 전형적 진입 조건

### 롱 진입 (예시) — "POC 첫 터치" 셋업
```
조건 1: 7일 누적 POC 계산 완료
조건 2: 가격이 POC 위에서 움직이다가 POC 레벨로 하락
        (POC가 아래 지지로 작동할 가능성)
조건 3: POC 첫 터치 시 거래량 정상~감소
        (과도한 매도세 아님 = 지지 가능)
조건 4: POC 터치 캔들의 꼬리가 아래로 길게 (매도세 흡수)
조건 5: 다음 캔들이 상승 마감

진입: 다음 상승 캔들 종가 시 시장가 롱
SL:   POC - ATR×0.5 (POC 진짜 깨짐 확인)
TP1:  7일 VAH (Value Area High)
TP2:  다음 HVN 또는 7일 고점
```

### 롱 진입 (예시 2) — "LVN 통과 모멘텀"
```
조건 1: 가격이 LVN 구간 진입
        (거래량 공백 = 지지/저항 약함)
조건 2: 진입 캔들 거래량 급증 
        (강한 추진력)
조건 3: 방향: LVN 다음에 HVN이 위에 있음 (TP 목표 명확)

진입: LVN 진입 확인 후 시장가
SL:   LVN 시작점 - ATR×0.3
TP:   다음 HVN의 아래 경계
```

## 강점

- **매우 고품질 신호**: POC 첫 터치는 역사적으로 높은 성공률 (Dalton 연구 기준 65~70%)
- **명확한 레벨**: 진입가/SL/TP가 구조적으로 명확
- **기관 친화**: 대형 기관이 같은 도구를 씀 → 셀프 성취적 예언
- **시장 국면 판별 가능**: Balance / Imbalance 구분으로 적절한 전략 선택
- **VIX와 무관**: 저변동성, 고변동성 모두 대응 가능

## 약점

- **계산 복잡도**: Bybit API 미제공 → 직접 히스토그램 구성
- **데이터 의존성**: 최소 7일 데이터 필요 → 신규 상장 코인 배제
- **기간 정의 필요**: 24h/7d/30d POC 중 어느 것이 맞는지 판단 어려움
- **늦은 반응**: 너무 많은 확인 조건으로 진입 지연 → 움직임 놓침
- **저유동성 코인 배제**: 거래량 적은 알트는 분석 자체 불가

## 이지원이 침묵하는 상황

- 신규 상장 30일 미만 코인
- 거래량 급감 구간 (dead market)

- POC 계산 불가능한 데이터 부족 상황

## 이지원이 강하게 말하는 상황

- 가격이 7일 POC에 **첫 터치**하는 순간
- 명확한 Balance 구간에서 VAH/VAL 근접
- 가격이 LVN 구간을 관통하려는 순간
- 긴 횡보 후 갑작스런 volume spike
